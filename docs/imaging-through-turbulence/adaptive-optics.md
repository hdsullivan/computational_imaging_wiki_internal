# Adaptive Optics

This area covers the use of physical systems to correct for optical aberrations in real-time. A typical adaptive optics system uses a wavefront sensor to measure distortions and a deformable mirror to apply a correction, physically compensating for the aberration before the light reaches the final imaging sensor. This hardware-level correction is often complemented by computational post-processing to achieve diffraction-limited imaging.
